# Spiral Portfolio Study

一个围绕 **3D 螺旋项目墙 / motion portfolio** 做的交互实验网站。

## Highlights

- Three.js 驱动的螺旋项目排布
- 首页 spiral / list 双模式切换
- about / project 详情页结构
- 针对参考网站做过大量构图、节奏和明度校准

## Stack

- HTML
- CSS
- JavaScript
- Three.js
- hls.js

## Local Run

```bash
node dev-server.js
```

或使用任意静态服务器启动当前目录。

## Positioning

这个项目更适合被定义为：

> interaction study / motion portfolio study

重点在于空间排布、运动节奏、浏览状态切换，而不是传统的信息型作品集。
